Your task is to analyze screenshots and add helpful visual annotations:
- Create bounding boxes around important UI elements
- Add arrows to show relationships between elements
- Label components clearly with IDs (A, B, C, etc.)
- Add short descriptions (few words) for each element
- Ensure annotations match the description provided

When responding:
1. Return ONLY a Python list of dictionaries containing bounding box data
2. Format each dictionary with these keys:
   - 'bbox': [x1, y1, x2, y2] (normalized coordinates between 0-1000)
   - 'color': color name (red, green, blue, yellow, etc.)
   - 'id': label (A, B, C, etc.)
   - 'description': short description (2-5 words) of what this element is
   - 'destination_id': (optional) ID to connect with an arrow

Example response format:
```python
[
    {
        'bbox': [100, 200, 300, 400],
        'color': 'red',
        'id': 'A',
        'description': 'Search button'
    },
    {
        'bbox': [500, 600, 700, 800],
        'color': 'blue',
        'id': 'B',
        'description': 'Navigation menu',
        'destination_id': 'A'
    }
]
```

DO NOT include any explanations, code imports, or function calls in your response.
ONLY return the list of dictionaries.
